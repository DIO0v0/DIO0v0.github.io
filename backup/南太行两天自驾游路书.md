## 2 周六 · Day1 北上深入峡谷
> 路线逻辑：从宝上路北上深入太行峡谷核心，到最北端后折返南下至石板岩

| 时间 | 景点 | 亮点与建议 |
|------|------|------|
| 07:15 | 宝上路观景台 | 俯瞰宝泉水库全景，国家地理级风光。晨光最佳，30分钟。导航搜"宝上路" |
| 08:00 | 武家湾 ⚡️充电 | 早餐补给。壶关大峡谷停车楼有快充，建议充满（今天山路耗电大） |
| 09:15 | 叠溪峡 | 古石叠溪峡，奇石峡谷，免费景点，出片率高 |
| 10:15 | 小壶口瀑布 | 太行版壶口，需徒步20分钟。带好水和防晒，适合亲子 |
| 11:30 | 黑毛沟大峡谷 🚨必去 | **南太行精华！** 峡谷一线天，幽深壮观，全程高能，强烈推荐停留1小时+ |
| 13:00 | 太行音乐公路 | 路面特殊设计，开车经过会"唱歌"，孩子会觉得神奇。慢慢开体验 |
| 13:30 | 午餐 | 太行一号公路沿线农家乐，推荐尝山野菜、农家豆腐、土鸡 |
| 14:30 | 棋子山 + 猪八戒 | 天然怪石像猪八戒，免费拍照打卡。可快速游览 |
| 15:15 | 西溪二仙庙 | **《黑神话：悟空》取景地！** 宋金古建，文化打卡必去，30分钟 |
| 16:00 | 齐王寨挂壁公路 | 人工悬崖公路穿越，门票约27元/人。注意单行道，小心会车 |
| 17:00 | 黄崖底十八盘 | 18连弯盘山路，落差约1000米！落日时分最美，孩子会兴奋 |
| 18:00 | 下山南下 | 开往石板岩，约60-80km山路。注意夜间山区行车安全 |
| 19:30 | 石板岩小镇 🏨住宿 | 办理入住，推荐带充电桩的山景民宿 |

> ⚠️ 今日重点：10个景点+200km山路，节奏紧凑。如某个景点想多待，可跳过叠溪峡或棋子山（锦上添花型）。黑毛沟大峡谷和西溪二仙庙是核心，不可跳过。

## 3 周日 · Day2 平顺精华 → 返程
> 路线逻辑：石板岩周边平顺段精华景点，然后从林州高速回家

| 时间 | 景点 | 亮点与建议 |
|------|------|------|
| 07:00 | 石板岩日出 | 四面环山，日出绝美。在民宿院子或阳台就能看 |
| 08:00 | 早餐退房 | 石板岩小镇早餐店多，豆浆油条+山野风味 |
| 08:30 | 石板岩晨景 | 早晨安静，写生学生多，石板建筑出片。快速逛30分钟 |
| 09:15 | 虹霓村瀑布 🚨必去 | 高山飞瀑，落差极大，水雾弥漫，空气清新。摄影圣地 |
| 10:15 | 神龙湾挂壁观景台 🚨必去 | **360度俯瞰大峡谷！** 震撼视角，孩子必拍。观景台本身免费 |
| 11:15 | 岳家寨挂壁公路 🚨必去 | **南太行标志性景观！** 悬崖挂壁+云端村落。注意单行窄路 |
| 12:30 | 山顶午餐 | 岳家寨附近有农家饭，简单用餐1小时 |
| 13:30 | 太行天路 🚨必去 | **云端山脊公路驾驶！** 醉太行+天路巅峰打卡点，全程约2小时。最美驾驶体验 |
| 15:30 | 下山至林州 | 到林州市区准备返程 |
| 16:00 | 林州市区 ⚡️充电 | **返程前务必充满！** 386km高速，充满后单程无忧 |
| 17:00 | 出发返程 | 走安林高速+G4京港澳高速南下 |
| ~19:00 | 鹤壁服务区 ⚡️补电 | 视电量决定是否补电。充满出发一般到这里余60%+ |

> ⚠️ 返程充电策略：石板岩夜充+林州充满=两段保障。林州充满后386km高速约耗电65-70%，到家余30%左右，绰绰有余。鹤壁服务区仅作保险方案。

## 4 充电规划总览
1. **周五夜 · 武家湾/民宿**
   夜间慢充或快充至满，确保周六一天山路无忧
2. **周六上午 · 武家湾壶关停车楼**
   游览时充电，快充桩。最新建成的智慧停车楼，快充+慢充
3. **周六夜 · 石板岩民宿**
   住宿时充电至满。选带充电桩的民宿，提前确认
4. **周日下午 · 林州市区**
   返程前充满，386km高速需要

## 7 温馨提示
- 🚗 **山路驾驶**：宝上路、十八盘、挂壁公路均为盘山路，部分路段窄仅一车宽。SUV优于轿车，新手司机慎入。会车时靠山侧让行。
- 🔌 **充电App**：出发前下载高德地图+特来电/小桔充电/国家电网App，提前查好沿途充电桩位置。山区充电桩数量有限，切勿等到电量极低再充。
- ☁️ **天气准备**：山区早晚温差大（白天25度可能晚上只有15度），带一件薄外套。夏季可能有阵雨，备雨具。
- 🏨 **住宿预订**：石板岩民宿周末火爆，务必提前3-5天在美团/携程预订，选"有充电桩""可停车"的民宿。
- 🍲 **餐饮建议**：太行山区农家饭实惠，人均30-50元。推荐山野菜、农家豆腐、土鸡、烙饼。自带零食和水。
- 👨‍👩‍👧 **亲子提示**：黑毛沟大峡谷、小壶口瀑布适合孩子探索；音乐公路孩子会觉得神奇；石板岩夜景适合全家散步。注意峡谷边缘安全。
